# 미니 홈페이지
프로필 관리, 다이어리, 게시판, 다이렉트메세지 기능이 있는 미니 홈페이지
- 사용 기술
  - React.js : 프론트앤드 라이브러리
  - Redux : 상태 관리
  - Firebase : 실시간 데이터베이스 및 인증 서비스
- 프로젝트 기간: 2024.08
